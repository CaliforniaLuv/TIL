  수 많은 브라우저마다 지원하는 js(ES 버전) 혹은 css 등등 서로 상이하다.
  예를 들어 css 속성 중 transition이라는 속성에 파이어폭스는 -moz-transition
  사파리는 -webkit- 등등 웹 브라우저마다 접두사가 서로 다르므로 이를 각각 맞춰서 구현하는 것을 크로스 브라우징 이라고 부른다.
  익스플로어 같은 경우 또 버전마다 지원하는 기능들이 다르므로 프로젝트 작업을 하기전 어디까지 지원을 할 것인지 
  또한, 어떤 브라우저들 위주로 호환성을 맞춰줄 것인지 선택이 필요하다.
  
  크로스 브라우징 작업을 통해 어떤 css 속성이나 JS 기능들이 지원되는지 확인하기 위해선 MDN같은 사이트를 통해 확인할 수 있다.

  크로스 브라우징 테스트는 QA를 통해 디자인 스타일 요소 혹은 기능 작동 여부를 직접적으로 확인하거나,
  유료 크로스 브라우징 테스트 사이트를 통해 각 브라우저마다 확인하는 방법들이 있다.
