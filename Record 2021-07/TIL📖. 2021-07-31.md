 ### 1. 알고리즘
 
  
  - 약수의 합
  
  
  ```js
  function solution(n) {
    let sum = 0;
    
    for(let i = 1; i <= n; i++) {
        if(n % i === 0) {
            sum = sum + i
        }
    }
    return sum
  }
  
  ```
